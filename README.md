Kullanıcıların girmek istedikleri şehir için güncel hava durumu bilgilerini gösteren Android uygulaması.
----------

## Özellikler

- Şehir bazlı güncel hava durumu sorgulama  
- Sıcaklık, açıklama, nem ve hissedilen sıcaklık bilgileri
